# # Zadání bakalářské práce


1. Seznamte se s dostupnými daty projektu MRE v kontextu úlohy extrakce strukturované informace z lékařských zpráv
2. Prozkoumejte aktuální LLM modely a příbuzné nástroje. Vytipujte vhodné kandidáty pro vyzkoušení na řešené úloze
3. Navrhněte postup extrakce strukturované informace za pomoci jazykových modelů a způsob validace správnosti řešení
4. Implementujte prototyp daného řešení
5. Zhodnoťte dosažené výsledky a jejich kvalitu
